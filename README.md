# Quantum-Onco-Diagnostic
Quantum-Enhanced Oncology Diagnostic System

A hybrid classical-quantum machine learning pipeline optimized for Apple M4 Neural Engine acceleration.

Features
*Quantum Circuit: 4-Qubit Variational Quantum Classifier (VQC) via PennyLane.
*Classical Backbone: ResNet-18 for image feature extraction.
*Tech Stack: FastAPI, PyTorch, Tailwind CSS.

How to Run
1. Install requirements: `pip install pennylane torch fastapi uvicorn`
2. Start backend: `python3 main.py`
3. Open `index.html` in your browser.
